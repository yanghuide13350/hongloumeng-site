# Hero 背景视频

把一个视频文件放进这个目录，首页 Hero 大图区会**自动**用视频替换静态图片（无需改代码）。

> 目前已内置一段程序化生成的水墨空镜 `hero.mp4`（以「林黛玉·潇湘馆」大图缓推 + 水墨调色 + 漂移雾气 + 前后回放无缝循环，约 12s / 720×1056 / <1MB）。
> 重新生成：`bash scripts/make-hero-video.sh`（依赖 ffmpeg）。
> 若日后用 AI（如小云雀）产出更好的空镜，直接覆盖 `hero.mp4` 即可。

## 用法

1. 准备一段竖向（约 3:4.4）、安静、循环友好的水墨/园林空镜，比如潇湘馆竹影、落花、烛火。
2. 命名为 `hero.mp4`（或 `hero.webm`）放到本目录。
   - 若放多个文件，优先使用文件名含 `hero` 的；否则取目录里第一个。
   - 支持后缀：`.mp4` / `.webm` / `.mov`。
3. 重新构建：`npm run build`（或 `npm run dev` 本地预览）。

## 建议规格

- 时长 8–15s，无缝循环；分辨率 720p~1080p；体积尽量 < 5MB。
- 视频会 `autoplay muted loop playsinline`，移动端也能自动播放。
- 静态图 `林黛玉·潇湘馆` 会作为 `poster` 首帧与无视频时的回退。

> 没有视频时，首页会用静态海报 + 缓慢推拉镜（Ken Burns）效果，体验同样完整。
